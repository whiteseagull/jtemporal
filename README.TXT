This distribution zip file contains:

- README.TXT             : this text file

- LICENSE.TXT            : a copy of the GNU LGPL license

- jtemporal.jar          : the distribution binary library

- jtemporal-src.jar      : the jtemporal distribution source files

- jtemporaldemo-src.jar  : a tiny demo application

- javadoc                : the javadoc of the distribution files

- jtemporaltests.jar: the binary executable unit tests 
  (you need JUnit 3.8 or above, http://www.junit.org to run the unit tests)
  You start the tests by typing : 
      java -cp jtemporal.jar;junit.jar net.sf.jtemporal.AllTests

- jtemporaltests-src.jar : the tests source files  
  (you need JUnit 3.8 or above, http://www.junit.org to compile the tests files)

Please check the JTemporal site (http://jtemporal.sourceforge.net) for the latest version and for more information.
